# Finding Similar Scientific Abstracts

## What it does

Detects near-duplicate arXiv abstracts using k-shingling, MinHashing, and LSH on 50k articles (subset of 3m of initial amount).

## Files

- `main.ipynb` — main notebook with the full pipeline
- `report.pdf` — written report

## Dataset

Download from [Kaggle](https://www.kaggle.com/datasets/Cornell-University/arxiv).
